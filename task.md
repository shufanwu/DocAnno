# 表格标注工具任务书

## 任务描述
使用python作为后端，网页为前端实现一个表格标注工具，标注人员通过观察表格图片标注为html格式标签。

使用流程：
1. 用户指定要标注的表格图片所在目录
2. 工具读取目录下所有图片文件并展示在前端页面
3. 在目录下，可能存在与图片对应的html文件，如果存在，读取html文件内容，解析为excel表格，渲染在excel表格编辑区域。
4. 如果存在错误，用户可在excel中修改表格（如合并单元格、修改值等）。
5. 用户修改完成后，点击保存，将excel表格保存为excel文件，同时将excel转换为对应的html标签保存为本地文件。
6. 工具会自动切换到下一张图片，用户可继续标注。

页面组件：
1. 导航栏，包括上传目录按钮、保存、上一张、下一张按钮
2. 图片展示区域，展示当前标注的表格图片
3. excel表格编辑区域，展示当前图片对应的标注结果，用户可在excel中修改表格（如合并单元格、修改值等）。
